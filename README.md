Inventory Management System

Overview:
A Java-based Inventory Management System developed using Swing GUI and MySQL/JDBC. The system helps manage products, monitor inventory levels, generate reports, and automate supplier reorder requests.

Features:
- Product Management (Add, Update, Delete Products)
- Inventory Tracking
- Low Stock Monitoring
- Automated Reorder Requests
- Inventory Reports
- Client-Server Communication using Java Sockets
- MySQL Database Integration

Technologies Used:
- Java
- Java Swing
- MySQL
- JDBC
- Socket Programming
- Multi-threading

System Architecture:
The system follows a client-server architecture where inventory data is stored in a MySQL database and reorder requests are communicated through socket-based networking.

Project Structure:
- GUI Components
- Product Management Module
- Database Connection Module
- Supplier Reorder Module
- Low Stock Monitoring Module

- Object-Oriented Programming (OOP)
- Database Design and Integration
- Networking and Socket Programming
- Multi-threading
- Software Development using Java
